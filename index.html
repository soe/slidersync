This page demonstrates SliderSync functionality. This functionality is implemented as a jQuery plugin. Three javascript files are included in this page: jQuery (js/jquery-1.7.2.minljs), your personalized copy of jQuery UI (js/jquery-ui-1.8.21.custom.min.js), and SliderSync (js/jquery-ui-slidersync.js). jQuery UI theme is also included here: Smoothness (css/smoothness/jquery-ui-1.8.21.custom.css) which imports the entire jQuery UI CSS Framework. You can build your own custom theme here: http://jqueryui.com/. SliderSync also has a short stylesheet: slidersync.css.
  • Implemented as a jQuery plugin
  • Using jQuery UI
  • jQuery UI themes compatible
  • Multiple SliderSync(s) on a page
  • Minimal DOM usage for optimal performance
  • Respect min/max values of sliders and rollback changes if they exceed these limits
  • Vertical and horizontal sliders
  • Layout easily definable
  • Simple visual tests included

sliders

list of sliders' values for which sliders will be rendered

false or list of integers. default: false. example: [30, 20, 80]

vals

show values of sliders

false or top, bottom, left, right. default: false.

val_prefix

prefix string for values of sliders

false or string. default: false.

val_suffix

suffix string for values of sliders

false or string. default: false. example: %

range

bind slider to mininum or maximum.

min or max. default: min.

min

minimum value of slider

integer. default: 0.

max

maximum value of slider

integer. default: 100.

animate

animate slider handle

false or true. default: false.

orientation

orientation of sliders

horizontal or vertical. default: vertical.

get total value


var total = 0;
$('#demo1-ss .slider').each(function() {
  total += $(this).slider('value');
});	
alert(total);
